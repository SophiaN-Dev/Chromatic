# ✿ Chromatic.

A cute, minimal color toolkit — palette generator, color picker, image palette extraction, and a built-in color library, all in one file.

No API keys. No backend. No build step. Just vibes.

**[Live Demo →](https://sophian-dev.github.io/Chromatic/)**

---

## What it does

### 🎨 Palette Generator
Generate beautiful 5-color palettes across 7 modes — Pastel (default), Vibrant, Earth, Mono, Neon, Jewel, and Sunset. Lock colors you love, swap any swatch with the color picker, and save your favorites. Hit **spacebar** to generate endlessly.

### 🎨 Color Library
Browse a curated collection of **300+ named colors** organized into 9 families — Reds, Oranges, Yellows, Greens, Blues, Purples, Pinks, Browns, and Neutrals. From everyday names like Coral and Mint to design classics like Prussian Blue and Burnt Umber to gems like Amethyst and Citrine. Click any color to copy its hex, star it to save.

### 🔍 Color Picker
Fine-tune any color with HSL sliders, a hex input, or the **EyeDropper API** (pick any color directly from your screen in Chrome/Edge). See RGB + HSL values live, then save or copy.

### 🖼 Image Extract
Drop in any image (drag & drop, file browse, or **Ctrl+V paste**) and extract its dominant colors using Canvas pixel quantization. Everything happens client-side — your images never leave your browser.

### 💾 Save & Share
- Star individual colors to build a personal collection
- Save full 5-color palettes with timestamps
- Copy CSS variables with one click
- Share palettes via URL — colors are encoded in the link
- Light / Dark mode toggle (preference saved)
- Generation counter tracks your creative streak

---

## How to use

1. Open `index.html` in any browser
2. Press **spacebar** or click **Generate** to create palettes
3. **Lock** colors you like (🔒), then regenerate the rest
4. **Star** (★) individual colors to save them
5. Click any color to copy its hex code
6. Switch tabs to explore the **Color Library**, **Color Picker**, or **Image Extract**
